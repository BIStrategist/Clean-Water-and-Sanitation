# SDG 6 – Clean Water & Sanitation Awareness Website

**Timeline:** April 2025  
**Team Member Highlighted:** Student 1

---

##  Project Overview
This is an interactive educational website developed to support the **United Nations Sustainable Development Goal 6 (Clean Water & Sanitation)**.  
The website focuses on raising awareness about water conservation, sanitation issues, and community engagement through interactive design and accessible content.

---

##  Features

### Full Project (Group)
- Interactive, user-friendly interface
- Accessible design for all users
- Engaging visuals and educational content
- Team collaboration to manage development tasks and ensure timely delivery

### Student 1 Contribution
**Pages Developed:**
1. **Splash Screen** – engaging entry point to the website with smooth navigation and visuals  
2. **Volunteer Page** – encourages community participation with interactive elements  
3. **Content 1 Page** – provides educational content about water conservation and sanitation  

**Key Responsibilities:**
- Designed and implemented the layout for assigned pages
- Developed interactive features using JavaScript
- Ensured responsive and accessible UI design
- Documented design decisions and development workflow for assigned sections
